Launchpadding Softwares

Ableton Live10 Suite is the main software I use to play covers. It provides audio feedback, as well as supporting Max4Live (M4L) devices. Live 10 Lite comes with every launchpad, but I use the Suite version which provides more features and solutions.

Apollo Studio powers the lights on the launchpad. It's a completely different engine from Ableton, but still keeps familiriaties with it. Gives acces to every color available on the RGB supported launchpads, no need of any top lights plugin, etc.

Max4Live Devices

These devices are only compatible with the Suite version of Ableton Live. Not neccesary to have installed but they makes producing a lot easier for me.
